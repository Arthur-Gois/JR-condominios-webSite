# JR-condominios-webSite
Sistema de gestão condominial para síndicos: controle de moradores, comunicados, reservas de áreas comuns e gestão administrativa.
🏢 Condominium Management System
Sistema web focado na demonstração de informações de um sindico de condominio.

🚀 Funcionalidades
Passar informações de contatos e redes sociais à frente

🛠️ Tecnologias
Este projeto foi desenvolvido utilizando:

Frontend: HTML5, CSS3 e JavaScript.

Arquitetura: Layout responsivo focado em usabilidade do usuario.
